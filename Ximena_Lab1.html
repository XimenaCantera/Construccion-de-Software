¿Cuál es la diferencia entre Internet y la World Wide Web? Internet es una red y World Wide Web es una colección de páginas. Se usa internet para acceder a la web.
¿Cuáles son las partes de un URL?
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? GET: Se usa para solicitar a los navegadores información que se encuentra en un servidor.
HEAD: Similar a GET pero no obtiene el cuerpo de la respuesta.
POST: Adjunta datos según lo solicitado, un ejemplo es cuando se requiere completar alguna información de un formulario.
PUT: Crea o cambia un elemento según la solisitud realizada.
PATCH: Se utiliza para cambiar temporalmente a un elemento.
DELETE: Borrar un elemento en espedífico.
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? POST ya que es el que se suele usar en los formularios para mandar la información.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?		 GET ya que es necesario solicitar el servidor web a través de la URL.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?		 Significa que la solicitud hecha por el usuario es correcta sin errores.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.


Son desaprobados cuando ya no son necesarios al tener nuevos atributos. Algunos ejemplos son:
  • acronym: Sirve para las abreviaciones. Ahora se usa abbr.
  • applet: Permte agregar un applet en un documento HTML. Ahora se usa object.
  • isindex: Manda información al servidor, pero ya no se usa gracias a la aparición de los formularios.
  • dir: Se usa para las listas, se secomienda usar ul.
¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
  • HTML5 necesita colocar el emento DOCTYPE declarado al inicio de la página para estar seguros de que el navegador compile la página en modo estandard.
  • HTML 5 ha actualizado con nuevos elementos de los que ya tenía en la HTML4.
  • Hay elementos de HTML5 que ya no son compatibles con HTML4.
¿Qué componentes de estructura y estilo tiene una tabla?

ESTRUCTURA:Una tabla tiene la etiqueta principal de table, para sus celdas se usa td o th y sus filas tr. Tambinén se pueden organizar en grupos de filas usando thead, tbody o tfoot. La tabla puede tener una leyenda usando caption y hacer referencia a las columnas.

Estilo: Dentro de un documento CSS se puede usar border-collapse para colocar bordes a la tabla, contable-layout: fixed, puedes dar a las columnas un tamaño de acuerdo con el ancho de sus celdas de encabezado.

Comentarios:
  • Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?
    Suena tentativo pero creo que se debería de evitar ya que al poner este tipo de condicionales en los estilos puede hacer que la forma que debe de tener un estilo sea mala o no tenga la estructura adecuada.
  • Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?
    Si no se escoge una foto concuidado puede hacer que la página se vea fea, desde el tamaño, los colores, la imagen en si puede que sea dificil leer el texto o que pierda la seriedad de la página. También las propiedades que se le ponga a la imagen, si quiere que se vea solo en una parte o que vaya "bajando" conforme el usuario lo haga.
  • Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?
    Preferería por puntos ya que tiene un mejor manejo de la dimensión del tamaño de las cosas, si se usan píxeles puede tener variaciones entre el tamaño de las letras ya que no todos tienen la misma cantidad de píxeles.
  • ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
    No, las versiones están por algo, tal vez mejoraron errores que tenían, tener versiones mejoradas ayudan a que algunos archivos pesen menos y mientras menor sea el peso mayor será la carag de la página.
